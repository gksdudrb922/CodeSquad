# CodeSquad
3단계 루빅스 큐브 구현하기
코드 실행 시 무한루프를 통해 명령어를 입력받는다.(URLDFB…’) 

Q가 입력받아지면 루프를 탈출한다. 

Random 명령어를 입력하면 100번의 무작위 셔플을 통해 큐브를 섞고 그 다음 명령어를 받도록한다.

문제에서 요구된 명령어들(U R L D F B+ ‘ or 2)을 입력하면 순차적으로 입력한 명령어에 맞게 수행한 후 큐브를 출력한다.

이 외의 명령어를 입력하면 오류메시지와 함께 재입력 하도록 한다.

만약 처음 큐브를 변경한 후 다시 모든 면을 맞추게 되면 축하 메시지와 함께 루프를 탈출한다.
